Interface SetOptions

An options object that configures the behavior of set() calls in firebase.firestore.DocumentReference.set DocumentReference, firebase.firestore.WriteBatch.set WriteBatch and firebase.firestore.Transaction.set Transaction. These calls can be configured to perform granular merges instead of overwriting the target documents in their entirety by providing a SetOptions with merge: true.

interface SetOptions {
    merge?: boolean;
    mergeFields?: (string | FieldPath)[];
}
